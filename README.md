# Car Damage Detection System

AI-powered web application for detecting vehicle damage from images using YOLOv8.

## Technologies

- Frontend: React + Vite
- Backend: FastAPI
- AI Model: YOLOv8
- Database: SQLite
- Authentication: JWT

## Features

- Vehicle damage detection
- Upload and analyze images
- Detection history
- Dashboard statistics
- Admin authentication

## Run Backend

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## Run frontend
```bash
npm install
npm run dev
```
